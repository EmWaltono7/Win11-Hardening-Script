Write and Test a Basic PowerShell Script

Completion requirements

🧪 Lab 3: Write and Test a Basic PowerShell Script

🎯 Objective

Students will write a PowerShell script that gathers basic system information—specifically the Windows version and computer hostname—and outputs it to the terminal. This lab ensures students can write, run, and verify a script without modifying the system.

📄 Part 1: Open Your Project

1. Launch Visual Studio Code

  • Open the folder: Win11-Hardening-Script

2. Open the Script File

  • In the Explorer sidebar, click on harden.ps1

✍️ Part 2: Write the Script

1. Add the Following PowerShell Commands

# Display the computer's hostname
Write-Host "Computer Name: $env:COMPUTERNAME"

# Display the Windows version
Write-Host "Windows Version:"
Get-ComputerInfo | Select-Object -Property WindowsProductName, WindowsVersion, OsHardwareAbstractionLayer

2. Save the File

  • Press Ctrl+S or go to File > Save

▶️ Part 3: Run the Script

1. Open the Integrated Terminal

  • Press `Ctrl+`` or go to View > Terminal

2. Run the Script

In the terminal, type:

.\harden.ps1

✅ You should see output similar to:

Computer Name: STUDENT-PC
Windows Version:
WindowsProductName        : Windows 11 Pro
WindowsVersion            : 10.0.22621
OsHardwareAbstractionLayer: 10.0.22621.1

🧪 Part 4: Test and Verify

1. Confirm Output

  • Ensure both hostname and Windows version details are printed
  • No errors should appear in the terminal

2. (Optional) Add a Timestamp

To show when the script was run, add this line at the top:

Write-Host "Script Run Time: $(Get-Date)"

📝 Submission Instructions

Students must:

  • Take a screenshot of the terminal showing successful script output
  • Submit the screenshot along with a copy of their harden.ps1 file
